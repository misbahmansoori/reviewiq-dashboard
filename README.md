# ReviewIQ – Ecommerce Review Intelligence Dashboard

## Overview

ReviewIQ is a modern SaaS dashboard designed to help ecommerce brand owners analyze and manage customer reviews received across multiple platforms such as Amazon, Flipkart, Meesho, and Shopify.

The platform transforms thousands of customer reviews into actionable insights through analytics, sentiment tracking, AI-generated recommendations, and reporting tools.

This project was created as part of the Filesure (Sellergrid) MERN Stack Developer Internship Assignment.

---

## Problem Statement

Ecommerce brands receive thousands of customer reviews every month across multiple marketplaces.

Manually reading and analyzing these reviews is time-consuming and inefficient.

ReviewIQ provides a centralized dashboard that helps businesses:

* Monitor customer sentiment
* Identify recurring complaints
* Discover positive feedback trends
* Generate AI-powered recommendations
* Export business reports

## Installation

Clone the repository:

```bash
git clone <repository-url>
```

Navigate to the project:

```bash
cd frontend
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```
